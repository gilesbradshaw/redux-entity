# redux-entity

A redux module allowing CRUD with real time updates

## Install

````
npm i redux-entity
````

## Clone

````
git clone https://github.com/gilesbradshaw/redux-entity.git
cd redux-entity
npm install
````

## Test

````
npm run test
````

## Lint

````
npm run lint
````

## Lint and fix

````
npm run lint:fix
````

## Build

````
npm run build
````
